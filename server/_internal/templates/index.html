Как запускать и работать

1. Запустите сервер
2. Подключите клиента

Запустите клиент: python client.py --client-id workstation-01 --server ws://{IP_WIFI}:8000/ws.

3. Работайте через UI

Откройте /processes для отчетов или /command для ручного запуска команд.
